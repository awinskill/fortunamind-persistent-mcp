## üéØ What is FortunaMind Persistent MCP?

FortunaMind Persistent MCP Server extends the standard MCP (Model Context Protocol) with persistent storage capabilities, enabling:

- **üìù Persistent Trading Journal** - Your entries survive across sessions
- **üîí Privacy-First Design** - Email-based identity, zero account data storage
- **üìä Tier-Based Access** - Flexible subscription tiers for different needs
- **üåê Multi-Platform Support** - HTTP API, Claude Desktop, and custom integrations
- **‚ö° Production-Ready** - Deployed on Render with Supabase PostgreSQL backend

## üõ°Ô∏è Privacy & Security

### Why Email-Based Identity?

Unlike other systems that store account information, we use a **privacy-first approach**:

- ‚úÖ **Your email generates a stable user ID** that survives API key rotations
- ‚úÖ **Raw emails are never stored** - only SHA-256 hashes with namespaces
- ‚úÖ **Zero account data storage** - no Coinbase account IDs, no API keys persisted
- ‚úÖ **Row Level Security (RLS)** - database enforces user isolation
- ‚úÖ **Works across exchanges** - same identity for Coinbase, Binance, etc.

### Gmail Users - Special Handling

We automatically normalize Gmail addresses:
- `john.doe+trading@gmail.com` ‚Üí same as `johndoe@gmail.com`
- `john.doe@gmail.com` ‚Üí same as `johndoe@gmail.com`
- This ensures consistent identity even with Gmail's aliasing

### Analytics and Reporting

```python
async def generate_monthly_report():
    async with FortunaMindPersistentClient() as client:
        
        # Get all entries from this month
        entries = []
        offset = 0
        
        while True:
            batch = await client.get_journal_entries(limit=100, offset=offset)
            if not batch.get('entries'):
                break
            entries.extend(batch['entries'])
            offset += 100
        
        # Analyze entries
        trades = [e for e in entries if e.get('entry_type') == 'trade']
        analyses = [e for e in entries if e.get('entry_type') == 'analysis'] 
        reflections = [e for e in entries if e.get('entry_type') == 'reflection']
        
        report = f"""
        üìä Monthly Trading Report
        
        ‚Ä¢ Total Entries: {len(entries)}
        ‚Ä¢ Trades Logged: {len(trades)}
        ‚Ä¢ Analyses: {len(analyses)}  
        ‚Ä¢ Reflections: {len(reflections)}
        
        Most Analyzed Symbols:
        """
        
        # Find most analyzed symbols
        symbols = {}
        for entry in entries:
            metadata = entry.get('metadata', {})
            if isinstance(metadata, str):
                try:
                    metadata = json.loads(metadata)
                except:
                    continue
            
            symbol = metadata.get('symbol')
            if symbol:
                symbols[symbol] = symbols.get(symbol, 0) + 1
        
        for symbol, count in sorted(symbols.items(), key=lambda x: x[1], reverse=True)[:5]:
            report += f"        ‚Ä¢ {symbol}: {count} entries\n"
        
        # Store the report
        await client.store_journal_entry(
            entry=report,
            entry_type="monthly_report",
            tags=["report", "monthly", "analytics"]
        )
        
        return report
```
